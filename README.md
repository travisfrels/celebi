# Celebi

A desktop probability calculator for the Trench Crusade tabletop miniatures wargame. Configure a dice pool scenario (pool size, top/bottom selection, pick count, modifier) and instantly see the cumulative probability of achieving each possible result or higher.

## Requirements

- Python 3.10 or later
- No external dependencies (uses Tkinter, which is included with Python)

## Usage

```bash
python main.py
```

## Running Tests

```bash
pytest
```
